# Shortcuts

Elementary defines the following shortcuts:

- Command palette - `ctrl+p`, `ctrl+shift+p`, or `esc`
- Open file - `ctrl+o`
- Open recent - `ctrl+shift+o`
- Save file - `ctrl+s`
- Save as - `ctrl+shift+s`
- New file - `ctrl+n`
- Toggle preview - `ctrl+e`
- Bold - `ctrl+b`
- Italic - `ctrl+i`
- Link - `ctrl+k`
- Uppercase - \`ctrl+u\`-
- Lowercase - \`ctrl+l\`
- Duplicate line - \`ctrl+d\`
